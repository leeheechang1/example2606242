# 동물 퀴즈 게임 기획서

## 1. 개요

| 항목 | 내용 |
|---|---|
| 제품명 | 동물 퀴즈 게임 (Animal Quiz) |
| 한 줄 소개 | 동물에 대한 재미있는 상식 5문제를 풀며 배우는 퀴즈 게임 |
| 대상 사용자 | 초등학생 및 동물을 좋아하는 누구나 |
| 형식 | 5문제 · 4지선다 · 정답 즉시 확인 · 해설 제공 · 총점 표시 |
| 실행 환경 | 단일 HTML 파일, 웹 브라우저에서 바로 실행 |
| 구현 파일 | `AnimalQuiz.html` |

---

## 2. 목표

- 동물에 대한 재미있는 상식을 **퀴즈 형식**으로 자연스럽게 학습
- 정답/오답 즉시 피드백 + 해설로 **능동적 학습** 유도
- 점수와 등급 시스템으로 **성취감과 재도전 동기** 제공
- 초등학생도 쉽게 사용할 수 있는 **귀여운 UI**

---

## 3. 화면 흐름

```
[시작 화면] → [문제 1] → [문제 2] → ... → [문제 5] → [결과 화면]
```

### 3-1. 시작 화면

- 사자 이모지 (바운스 애니메이션)
- 타이틀: "동물 퀴즈"
- 설명: "동물에 대해 얼마나 알고 있나요? 5개 문제를 풀어보세요!"
- **시작하기** 버튼 (보라-핑크 그라데이션)

### 3-2. 문제 화면

- **진행 바** — 현재 문제 번호 / 5, 누적 점수
- **문제 카드** — 동물 이모지 + 문제 번호 + 질문 텍스트
- **4개 선택지** — A/B/C/D 원형 번호 + 텍스트
- 선택 후:
  - 정답 → 초록 하이라이트 + ✅
  - 오답 → 선택한 보기 빨강 + 정답 보기 초록
  - **해설 카드** 표시 (💡 알고 있었나요?)
  - **다음 문제** 버튼 표시

### 3-3. 결과 화면

- 등급별 이모지 + 타이틀
- **총점** (100점 만점)
- **별 등급** (⭐ 1~5개)
- 격려 메시지
- **다시 풀기** 버튼
- 80점 이상 시 꽃가루(confetti) 효과

## 4. 문제 콘텐츠

| # | 이모지 | 질문 | 선택지 | 정답 | 해설 |
|---|---|---|---|---|---|
| 1 | 🐋 | 세상에서 가장 큰 동물은? | A. 아프리카 코끼리 / B. 대왕고래 / C. 기린 / D. 공룡 | B | 길이 약 30m, 무게 약 150톤. 심장이 소형 자동차 크기 |
| 2 | 🦒 | 기린의 목뼈는 몇 개? | A. 20개 / B. 14개 / C. 7개 / D. 뼈가 없어요 | C | 사람과 같은 7개, 뼈 하나가 약 25cm |
| 3 | 🐙 | 문어의 심장은 몇 개? | A. 1개 / B. 2개 / C. 3개 / D. 8개 | C | 주심장 1개 + 아가미심장 2개 |
| 4 | 🐨 | 하루에 가장 많이 자는 동물은? | A. 고양이 / B. 나무늘보 / C. 코알라 / D. 하마 | C | 하루 22시간 수면, 에너지 절약 |
| 5 | 🦇 | 유일하게 날 수 있는 포유류는? | A. 날다람쥐 / B. 박쥐 / C. 슈가글라이더 / D. 날여우 | B | 날다람쥐는 활공, 박쥐만 진정한 비행 |

## 6. 인터랙션 규칙

- 선택지를 고르면 **즉시 정답/오답 표시** — 재선택 불가
- 정답: 해당 보기 초록 하이라이트
- 오답: 선택한 보기 빨강 + 정답 보기 초록
- 나머지 보기는 흐리게 비활성화
- 해설과 "다음 문제" 버튼은 선택 후에만 등장
- 마지막 문제에서는 "결과 보기 🎉" 버튼으로 변경

## 9. 기술 구성

- **형식**: 단일 HTML 파일 (`AnimalQuiz.html`)
- **상태 관리**: `current` (현재 문제), `score` (점수), `answered` (답변 여부)
- **데이터**: `quizData` 배열 — 이모지, 질문, 선택지, 정답 인덱스, 해설
- **애니메이션**: CSS `@keyframes` — fadeUp(화면 전환), bounce(시작 이모지), confettiFall(꽃가루)
- **배포**: 단일 HTML 파일, 인터넷 없이 실행 가능
